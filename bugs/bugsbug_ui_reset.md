# BUG-008: Сброс состояния интерфейса при перезагрузке страницы

**Описание:** Приложение не сохраняет введенные данные (URL, Headers, Body) при случайном нажатии Ctrl+R или обновлении страницы.

**Шаги воспроизведения:**
1. Ввести любой URL и заполнить тело запроса.
2. Нажать клавиши Ctrl + R.

**Фактический результат:** Страница обновляется, все поля ввода очищаются.
**Ожидаемый результат:** Появление модального окна с предупреждением о потере данных или сохранение данных в кэше браузера.
