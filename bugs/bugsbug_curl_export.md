# BUG-009: Отсутствие функции экспорта в cURL

**Описание:** Кнопка cURL в верхней панели работает только на импорт. Невозможно скопировать текущий настроенный запрос для использования в терминале.

**Шаги воспроизведения:**
1. Настроить любой запрос (метод, URL, тело).
2. Нажать кнопку "cURL".

**Фактический результат:** Открывается окно только для вставки (Import).
**Ожидаемый результат:** Наличие кнопки "Copy as cURL" или автоматическая генерация строки запроса.
